このサイトは以下の3ファイル構成です。

| ファイル | 役割 | 置き場所 |
|---|---|---|
| index.html | サイト本体 | GitHubリポジトリ minna-aso のルート |
| code.gs | 会員DBバックエンド | Googleスプレッドシートの Apps Script |
| このガイド | 手順書 | （アップロード不要） |

## STEP 1：スプレッドシート＋GASの準備（10分・初回のみ）

1. https://sheets.google.com で新しいスプレッドシートを作成
   （名前は「みんあそくん会員DB」など自由）
2. メニュー「拡張機能」→「Apps Script」を開く
3. 最初から入っているコードを全て消し、**code.gs の内容を全部貼り付け**て保存（Ctrl+S）
4. 右上「**デプロイ**」→「**新しいデプロイ**」
   - 歯車アイコン → 種類:「**ウェブアプリ**」
   - 説明: 自由（例: v1）
   - 次のユーザーとして実行:「**自分**」
   - アクセスできるユーザー:「**全員**」 ← ここが重要
5. 「デプロイ」を押す（初回はGoogleアカウントの承認画面が出ます。
   「詳細」→「（プロジェクト名）に移動」→「許可」と進めてください）
6. 表示された「**ウェブアプリのURL**」をコピー
   （`https://script.google.com/macros/s/XXXX/exec` の形式）

> シート（Members / Ideas / Events / Help / Memories）は
> 初回アクセス時に自動で作られます。手動作成は不要です。
> 会員が登録するたびに Members シートに行が増え、
> **スプレッドシートを開くだけで会員情報を確認できます。**

## STEP 2：index.html にURLとキーを記入

index.html 内の「設定」ブロック（ファイル後半の `<script>` 冒頭）にある2行を書き換えます。

```javascript
const GAS_API_URL = "https://script.google.com/macros/s/XXXX/exec";  // STEP1のURL
const GMAPS_API_KEY = "AIzaSy...";  // Maps APIキー（任意）
```

- GAS_API_URL が空のままだと「端末内デモモード」で動作します（動作確認用）
- GMAPS_API_KEY のセキュリティ設定は「APIキー設定ガイド.md」参照
  （リファラー制限: `https://kenken6291.github.io/*`）

## 機能一覧

### 会員まわり
- **登録**: ニックネームを入れるだけで8桁の会員番号を自動発行（重複なし）
- **ログイン**: 8桁番号を入力。同じ端末なら次回から自動ログイン
- **管理者**: 会員番号 **19626291** でログインすると「🛡管理者」タブが出現
  （初回ログイン時に管理者アカウントが自動作成されます）

### 管理者パネル
- 会員・思い出・アイデア・イベント・助け合いを一覧表示
- **チェックボックスで複数選択 → 一括削除**（先頭のチェックで全選択）
- ✏️ボタンで投稿内容の個別編集
- 削除・編集はスプレッドシートに即反映
- 管理者自身（19626291）は削除できない安全装置付き

### 場所と思い出（統合タブ）
- 🗺️ **あの場所の今**: 検索 → 地図＋ストリートビュー → お気に入り保存
- 📷 **思い出くらべ**: 昔の写真×現在の風景を「並べて／スライダー／透過」で比較
  （写真は端末内処理のみ。サーバーには送信されません）
- 📖 **みんなの思い出**: 場所・年代・ひとことをみんなで共有
  → 「ここを一緒に歩きませんか」からイベント企画へ直結

## セキュリティ設計（重要）

このサイトのAPIは以下の方針で保護されています。

**1. 管理者番号はサーバー側にのみ存在**
- index.html（公開ソース）には管理者番号を書いていません
- 管理者判定はGAS内部でのみ行われ、画面の管理者タブはサーバーの応答（isAdmin）でのみ出現します
- 管理者番号を変更したい場合：Apps Script画面 → ⚙「プロジェクトの設定」→「スクリプト プロパティ」→ プロパティ名 `ADMIN_NO`、値に新しい8桁番号を登録（コード修正・再デプロイ不要）
- **公開運用を始める前に、初期値から別の番号へ変更することを強く推奨します**

**2. 会員番号は他人に渡らない**
- 会員番号はログインの鍵（パスワード相当）です
- APIは他人の番号を `*****291` のようにマスクして返し、生の番号は本人と管理者にしか渡しません
- 「自分の投稿かどうか」はサーバーが付けるフラグで判定するため、番号の照合をブラウザ側で行いません

**3. 入力値はサーバー側で制限**
- ニックネーム20字、投稿500字など全フィールドに上限があり、巨大データによる荒らし・シート破壊を防ぎます

**4. 権限チェックはすべてサーバー側**
- 削除・編集は「本人または管理者」の検証をGAS内で実施。ブラウザの表示を改造しても他人の投稿は消せません
- 管理者アカウント自体は一括削除の対象から除外されます

**注意して運用すべきこと**
- 管理者番号は絶対に人に教えない・画面を見せるときは管理者タブを開かない
- 会員には「会員番号は合鍵と同じ。人に教えないで」と案内してください
- GASのURL自体は公開されても問題ない設計ですが、心配な場合は再デプロイでURLを変更できます

## コードを更新したときの再デプロイ

GAS側（code.gs）を修正した場合:
「デプロイ」→「デプロイを管理」→ 鉛筆 → バージョン「新バージョン」→「デプロイ」
（URLは変わらないので index.html の修正は不要です）

## よくあるトラブル

**Q. 登録ボタンを押すと「通信エラー」になる**
A. GASのデプロイ設定で「アクセスできるユーザー」が「全員」になっているか確認してください。「自分のみ」だと外部から呼べません。

**Q. スプレッドシートに行が増えない**
A. デモモード（GAS_API_URL が空）で動作しています。STEP 2 のURL記入を確認してください。

**Q. 管理者タブが出ない**
A. 19626291 でログインし直してください。一般会員の画面には表示されません。
